Java简介

Java历史

什么是java

1995年，Sun公司(Stanford University Network)推出的面向对象语言，最初名字OAK。
JAVA是一种通过解释方式来执行的语言，语法规则与C++类似。可以跨平台。

Java版本

  1. Java SE

    Java的标准版本，用于桌面应用程序开发，是JAVA基础，包含JAVA语言基础，JDBC（JAVA数据库连接）、IO、多线程技术

包含版本：JVM、JRE、JDK和Java语言

  • JVM:Java Virtual Machine
  • 将Java编译好的文件翻译成对应的机器语言
  • Jre:Java SE Runtime Environment
  • 包含用于Java 的运行环境
  • JDK:Java Development kits
  • 包含用于Java开发的工具
  1. Java EE

    Java EE是Java企业版，用于开发企业分布式网络程序，电子商务网站和ERP（企业资源规划）系统，核心EJB（企业Java组件模型）

  2. Java ME

    Java ME主要应用于嵌入式系统开发，需要虚拟机和底层操作系统支持，所以被时代淘汰，Android应运而生。

    Java API 文档

  • API全称是Application Programming Interface，应用程序接口，API文档是Java程序开发过程中不可或缺的编程词典。记录了JAVA语言中海量的API，包括类的继承结构，成员变量，构造方法，静态成员的描述信息和详细的说明等内容。

    JDK配置（待补充）

  • bin:binary(二进制) 存放了Java的可执行文件
  • lib:存放了Java类库文件
  • include:存放了用于本地方法的文件
  • demo：存放演示程序
  • jre：存放JAVA运行环境

    语法（主要阐述与C++不同之处）

    与C++大致相同，学起来比较快，但是要注意很多从前的编码习惯，所以主要突出于C的不同之处

    常量

C中常量的定义为#define或者const,java中不仅需要指定数据类型，还需要通过final关键字进行限定。
